# edgar.net
.NET 10 API for utilizing the SEC's EDGAR database.

## Features
- EDGAR search and filing retrieval.
- Implementations for:
  - Extracting buyout prices from DEFM14 (definitive merger) filings.
  - Extracting transactions from Form 4 (insider trading) filings.
- Simple caching implementation to reduce request burdern.
